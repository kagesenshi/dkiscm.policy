dkiscm.policy Installation
--------------------------

To install dkiscm.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``dkiscm.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        dkiscm.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
